This year, we are using the Fat-Free Framework for PHP (https://fatfreeframework.com/), normally referred to here as F3. Some of these notes are older and do not take F3 into account. This is OK, in that all the normal PHP approaches still work, but you will not need to study (and should strictly avoid) the "vanilla PHP" method for connecting to databases, for example.
