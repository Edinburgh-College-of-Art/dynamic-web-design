Google Maps API

Page 1, Mike Greer, 26.02.2008

Google Maps API
Website http://code.google.com/apis/maps/
Key required? Yes
Free? When used on websites free to consumers.
Version: 2

The Google Maps API allows you to embed dynamic maps in your webpage. Using javascript, you can switch on and off various features of the map, place markers on it, or add an overlay to display extra information. As you might expect, it is brilliantly documented here.

Setting up a basic map takes minutes (the following uses Google's example code, supplied when you complete the first step):

  • Register for an API key - this key is specific to the site you wish to place the map on (so register the correct folder).
  • Bring in the map javascript via a script tag in the header:
    <script src="http://maps.google.com/maps?file=api&v=2&key=YOURKEY" type="text/javascript"></script>
  • Add a javascript function which will create the map:
    <script type="text/javascript">
    function initialize() {
    if (GBrowserIsCompatible()) {
    var map = new GMap2(document.getElementById("map_canvas"));
    map.setCenter(new GLatLng(37.4419, -122.1419), 13);
    }
    }
    </script>
  • Create a div in the body of your HTML as a placeholder for the map:
    <div id="map_canvas" style="width: 500px; height: 300px"></div>
